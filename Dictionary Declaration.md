# Dictionary Declaration

Dictionary in python are used to store key:value pair. For example, you could use to store details of student as shown below. 

Here are dictionaries Tom and Sheela

Tom = {\
  "Major": "Physics",\
  "Age": 22,\
  "Gender": "Male",\
}\
print(Tom)


Sheela = {\
  "Major": "Chemistry",\
  "Age": 21,\
  "Gender": "Female",\
}\
print(Sheela)
