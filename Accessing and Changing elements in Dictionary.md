# Accessing Elements in Dictionary

Elements in the dictionary could be accessed using the below command

Sheela = {\
"Major": "Chemistry",\
"Age": 21,\
"Gender": "Female",\
}\

Sheela["Major"]

Elements in a dictionary could also be accessed using .get()

Sheela.get("Major")
